AI-powered education platform where tutors upload syllabus PDFs via an Admin Dashboard. Students access a portal to chat with an AI tutor, view course documents in-browser, and take autogenerated MCQ quizzes based strictly on uploaded content. If information is unavailable, the AI replies “Not in syllabus.”
